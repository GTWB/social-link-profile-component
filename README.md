# Social Link Profile Component

A responsive “social links profile” card built as a Frontend Mentor challenge solution. The goal is to recreate the design as closely as possible and include clear hover/focus states for interactive elements.

## Demo

- Live site: https://react-social-component-profile.netlify.app/
- Frontend Mentor challenge: Social Links Profile (Frontend Mentor)

## Features

- Responsive layout (mobile → desktop)
- Accessible interactive states (hover + keyboard focus)
- Clean, reusable component structure
- Easy to customise links, avatar, name, location and bio

## Built with

- React (Create React App)
- HTML / CSS
- JavaScript

## Getting started

### Prerequisites

- Node.js (LTS recommended)
- npm (comes with Node)

### Install & run locally

# install dependencies

npm install

# start dev server

npm start
